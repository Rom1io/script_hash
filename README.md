# Script Hash

### Install :

Clone Repo Git: [Here](https://github.com/Rom1io/script_hash)

```zsh
% git clone https://github.com/Rom1io/script_hash.git
% cd script_hash
```
### Install dependencies :

```zsh
% yarn install
```
### Start script:

Add you're files in files folder

Start Script :  
```zsh
% node hash.js
```

View your csv file in the "result" folder.
